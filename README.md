# Команды бота

- `/start` — показывает статус доступа и модель. Если доступ активен, говорит об этом.
- `/help` — список команд.
- `/pay` — тестовый инвойс Stars (XTR). Если доступ уже есть, инвойс не отправляется.
- `/paytest` — тестовая активация без списания, сразу включает доступ.
- `/llama` — выбрать модель meta-llama/llama-3.3-70b-instruct:free.
- `/gpt` — выбрать модель openai/gpt-oss-20b:free.
- `/gemini` — выбрать модель google/gemini-flash-1.5-8b:free.
- `/admin_stats` — для admin_ids, краткая статистика пользователей/баланса/транзакций.
- `/admin_tx` — для admin_ids, последние транзакции Stars.

Поведение чата:
- После активации доступа можно просто писать текст, бот ответит выбранной моделью и сохранит историю (последние 5 сообщений на модель).
